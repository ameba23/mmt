# Set-up

Environment
-----------

```BASH
git clone http://code.constructiveproof.com/git/mmt/mmt.git && cd mmt
bundle
rake
```

Development Environment
-----------------------

`touch .env && echo APP_DOMAIN='blockades.dev' >> .env`
`rails s`
Navigate to [mmt.blockades.dev:9999](http://mmt.blockades.dev:9999/)
